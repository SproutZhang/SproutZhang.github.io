CSS3 - 选择器

CSS3中给了我们更多的选择器让我们来获取元素，极大程度提高了查找元素的精度以及准确性，值得庆幸的是绝大多数的选择器的语法跟jQuery中兼容

★ 属性选择器

  • 下列属性名为att
  1. E[att] : 应用att属性的所有标签
  2. E[att=”val”] :属性值为val的标签
  3. E[att~=”val”]:属性值用空格进行分隔，其中一个为val的标签
  4. E[att^=”val”]:属性值以val为开头的标签
  5. E[att$=”val”]:属性值以val为结尾的标签
  6. E[att*=”val”]:属性值中包含val的标签

★兄弟选择器

兄弟选择器的作用是：选择相邻的兄弟节点

  • 语法：选择器1~选择器2

    • 在相同的父元素中，只能选择选择器1之后的兄弟元素，中间有别的标签分隔也能选中

★ 伪类选择器

  1. E:first-child:第一个子元素
  2. E:last-child: 最后一个子元素
  3. E:nth-child(n):索引是从1开始，可以写n，n从0开始，表示第n个元素。可以传入表达式，比如2n，2n+1等，可以传入特殊字符：even(偶数),odd(奇数);
  4. E: nth-last-child(n):跟E: nth-child(n)类似，从后往前计算。

  5. E:nth-of-type(n):

    E:nth-child(n)括号中的索引包含除E以外的其他兄弟节点，所有的兄弟节点都会算进去。
    E:nth-of-type(n)：会考虑类型,只会算E的节点

    ① E:first-of-type : 第一个E节点
    ② E:last-of-type : 最后一个E节点

★ 伪元素选择器

伪元素选择器推出的初期，就是使用 ”: “一个冒号，这时很容易和伪类选择器搞混。所以在C3 中，为了跟伪类选择器区分，推荐使用“::” 两个冒号，但是一些版本比较老的浏览器，不支持 两个冒号的写法。所以，如果你想让格式看起来更加规范，就使用”::” ，如果项目需要考虑老 版本的浏览器，就使用”:” 。在这统一使用”::”。

  • 常用（重要）

    • E::before : 在双标签E内最前添加伪元素

    • E::after : 在双标签E内最后添加伪元素

    • 使用注意：

      • 1.必须用于双标签内
      • 2.必须设置content属性，内容可以为空
      • 3.生成的伪元素是一个行内元素，如果要设置宽高，需变为块级元素
      • 4.使用星号设置全部样式的时候，不包含用伪元素创建出来的标签

  • 不常用

    • E::first-letter : 首字母
    • E::last-line : 首行， 用户改变窗口尺寸的时候，始终是首行
    • E::selection : 设置被选中的样式，只能设置background-color, color, text-shadow这几种属性。

    • E::placeholder: 控制input输入框里的占位符的外观，但是这个选择器的兼容性并不是很好，使用时需要加前缀，写兼容。
