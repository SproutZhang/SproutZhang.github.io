CSS总结

css部分总结

一、选择器

  1. 标签选择器
  2. 类选择器
    • 一个标签可以调用多个类选择器

  3. id选择器

    • id必须保证唯一，一个标签可以同时调用类和id选择器，但是不能同时调用两个id选择器

  4. 通配符选择器 ：* 不推荐使用

  5. 交集选择器 ：div.box\div#box
  6. 后代选择器 ：空格
  7. 子代选择器 ：父>子
  8. 并集选择器 ： 逗号分隔

二、块、行内和行内块

2.1 区分

  1. 块级元素有哪些

    • div \ p \ h1-h6 \ ul li \ ol \dl
    • 独占一行、可以定义宽高、默认父元素的宽度（高度不会）

  2. 行内元素有哪些

    • span \ a \ em \ i \ b \ strong \ del
    • 在一行显示、不能直接定义宽高，内容撑开的，只能设置左右的内外边距，不能设置上下

  3. 行内块

    • img \ input \ textarea
    • 在一行显示，可以定义宽高，可以设置上下的内外边距

2.2 互转

  1. 块转行内： display : inline;
  2. 行内转块： display : block;
  3. 块、行内转行内块： display : inline-block;

三、css的三大特性

3.1层叠性

  • 当多个样式所用于同一个标签的时候，总是执行后边的样式

3.2继承性

  • 嵌套关系下，文字的所有属性都能被继承
  • 特殊情况：① h1-h6的粗细和文字大小自带的，不能继承 ； ② a标签颜色不能继承父元素

3.3 优先级

  • 默认样式 < 标签选择器 < 类选择器 < id 选择器 < 行内样式 < !important
  • 0 1 10 100 1000 1000以上

四、背景属性

  • background-position : left \ right \ top \ bottom center;

    • 直接写两个方位值，顺序无要求
    • 只写一个方位值，另一个默认居中
    • 写数值 第一个距左 第二个距顶

  • background-attachment(设置背景是否固定)

    • scrol 默认，滚动
    • fixed 固定不动，相对于浏览器
  • 背景属性连写：(背景图片为必写项)

五、盒模型

5.1什么是盒子模型

  • 盒子模型包括：内容(content)、外边距(margin)、内填充(padding)、边框(border)

5.2问题

  1. 垂直方向的margin合并

    • 出现情况：上下两个盒子、上盒子设置了margin-bottom: 30px; 下盒子设置了margin-top: 20px; 这时就会出现外边距合并
    • 只出现于垂直方向
    • 取大值

  2. 外边距塌陷

    • 两个盒子嵌套： 给子元素margin-top,父元素也会一起向下移动
    • 只出现于垂直方向
    • 解决： ①给父盒子设置边框 ②给父盒子overflow: hidden;

5.3盒子宽度问题

  • 盒子宽度= border + padding + width

六、浮动

元素浮动后会脱离标准流，不占据原来的位置，所以有时候在不给浮动元素设置高的情况下，下面的盒子会占据浮动盒子原来的位置，这时就要清除浮动，并不是不浮动，而是清除浮动带来的影响

  • 清除浮动的方法以及优缺点
  1. 额外标签法
  • 缺点：会产生多余的代码
  1. 给浮动元素父容器加：overflow: hidden
    • 缺点：溢出的部分会隐藏

  1. 单伪元素清除浮动法

    ​
    • 比较老一些的网站使用，例如：新浪，网易

  1. 双伪元素清除浮动法

    • 比较新的网站用，例如京东、小米、淘宝

七、css初始化

  • 作用：
    • 各浏览器对基础样式渲染不一致，css样式初始化为了考虑到浏览器的兼容问题，使渲染的页面风格统一

八、定位position

  1. 静态定位：position: static 默认
  2. 绝对定位：position: absolute
    • 基于浏览器出发、脱标、行内可转行内块
  3. 相对定位：position: relative
    • 基于自身出发、不脱标
  4. 固定定位：position: fixed
    • 脱标
