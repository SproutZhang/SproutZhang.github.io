三、自定义对象

  • 思路：封装一个构造函数，将来用new调用，创建一个空白对象，为它绑定属性和方法

四、面向对象和基于对象

4.1面向对象

可以创建自定义的类型、很好的支持继承和多态

  1. java
  2. c++
  3. c#

4.2基于对象

无法创建自定义类型、不能很好的支持继承和多态

  1. js是基于对象，基于Object

4.3JS的内置对象

  1. Array是JS的内置对象，里面有属性和方法

    • 属性：arr.length
    • 方法：
      • push() 末尾添加
      • pop() 末尾删除
      • unshift() 最前添加
      • shift() 最前删除
      • reverse() 翻转
      • sort() 排序

  2. String

    • 属性：str.length
    • 方法：
      • indexof() 给字符查索引
      • charAt() 给索引查字符
