JS基础—数据类型、操作符、流程控制

JS基础—数据类型、操作符、流程控制

一、数据类型

1.1简单数据类型

简单数据类型存放在栈，传值

  1. string
  2. number
  3. Boolen
    • true : 除0数字、字符串、object
    • false : 0、’ ‘ 、undefined、null、NaN
    • if在判断时会把括号里的值强行转为boolen类型进行判断
  4. undefined
    • undefined未定义，null定义了未赋值

1.2复杂数据类型

复杂数据类型存放在堆，传址

  1. object
  2. function
  3. Array
  4. Date
  5. RegExp
  6. Error
  7. …

1.3数据类型转换

  1. 转换为string

    • +
    • String() 构造函数法
    • 变量.toString() ; undefined和null 没有toString方法

  2. 转换为number

    • - * / (隐式类型转换)
    • Number(变量)
    • parseInt() ; 取整 parseFloat(); 取浮点数 （ 强制类型转换）

  3. 转换为boolen

    • Boolen()
    • !!变量（两次取反）

二、操作符

2.1算数运算符

  1. 一元运算符： 正号、负号、++、–、平方
  2. 二元运算符：+ - * /
  3. 三元运算符：值1?值2:值3

2.2 逻辑运算符

  1. ||
  2. &
  3. ！

2.3比较运算符

  1. >
  2. <
  3. ==
  4. =>

2.4赋值运算符

  1. =
  2. +=
  3. -=
  4. *=
  5. /=
  6. %=

2.5&&和||

  1. && ：有一个为false，结果就为false，如果第一个为false，则后面的不执行（短路运算），如果两个都为true，则取后面的。一旦找到false，立马返回
  1. ||：一旦找到true立马返回，两个都为false，取后

2.6数据类型的优先级

  1. ()
  2. !、-(负数)、++、–（一元运算符）
  3. *、/、%
  4. +、-(二元运算)
  5. <、<=、>、>=(一级逻辑运算)
  6. ==、!=、===、!==(二级逻辑运算)
  7. && (三级逻辑运算)
  8. ||
  9. ?:
  10. =、+=、-=、*=、/=、%= (赋值运算)

2.7 ==和===的区别

== 会自动类型转换，比较时只比较值，不比较类型
=== 全等，不会转换类型，不仅比较值，还比较数据类型

三、流程控制

3.1顺序结构

  • 从上而下，从左到右

3.2选择结构

  1. if
  2. switch

3.3循环结构

  1. for循环
  2. while
  3. do…while

3.4区别

  1. break和continue的区别：

    • break立即跳出循环，后面的代码不执行
    • continue：跳出本次循环，进入下次循环

  2. while和do…while循环区别：

    • while先判断条件，符合判断再执行里面代码
    • do…while 先执行一次，再判断
