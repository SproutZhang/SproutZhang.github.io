jQuery

jQuery

一、介绍

  1. 什么是jQuery?
  • 是JavaScript的框架库，就是一个JS文件，这个文件中封装了很多的兼容代码，封装了常用的函数，jQuery是函数的一种
  • jQuery并不能把所有的兼容都解决，浏览器的兼容不仅仅是JS还有css，jQuery封装的还不是很完美，所以有些功能或者浏览器的兼容问题不能都通过jQuery来解决，只能通过原生JS
  1. 常见的框架库有哪些？
  • Prototype、YUI、Dojo、ExtJs、jQuery
  1. 顶级对象
  • DOM中的顶级对象是document
  • BOM中的顶级对象是window
  • jQuery中的顶级对象是$\jQuery
  • $是jQuery的标识符，$===jQuery
  1. 页面加载的几种方式

  • DOM中的页面加载

  • jQuery中页面加载的方式

二、选择器

2.1普通选择器

  1. $(‘#id’)
  2. $(‘标签的名字’)
  3. $(‘.class’)
  4. $(‘li.class’)
  5. 多条件选择器 $(‘p,span,div,.cls’)

2.2层级选择器

  1. $(“div span”) 空格 div后的所有的span（子子孙孙）
  2. $(“div>span”) > div中所有是span的直接子元素（子）
  3. $(“div~span”) ~ div后的所有兄弟span元素
  4. $(“div+span”) + div后紧邻的第一个span兄弟元素

2.3 索引选择器

  1. $(“#uu>li: eq(3)”) :eq(索引) 获取指定索引位置的元素
  2. $(“#uu>li: gt(3)”) :gt(索引) 获取索引大于…的元素
  3. $(“#uu>li: lt(3)”) :lt(索引) 获取索引小于…的元素
  4. $(“#uu>li: gt(5):lt(3)”) 先找所有大于5的，再在大于5的里找小于3的

三、方法的总结

  1. .text() 方法和innerText一样
  2. .html() 方法和innerHTML一样
  3. .val() 方法和value属性一样
  • 以上三个方法，括号里不写内容是获取，写内容是设置
  • 样式方法
  1. .css() 方法
    4.1 .css(“width”,”30px”) 逗号隔开
    4.2 .css({“width”:”30px”,”height”:”40px”}) 键值对形式 设置多个
  2. .addClass() 为某一元素添加类样式，可添加多个，用空格隔开
  3. .removeClass() 移除类样式，可移除多个，用空格隔开，如果内部没有任何参数，则移除所有样式
  4. .hasClass() 判断当前元素是否应用了指定类样式，返回值布尔类型
  5. .toggleClass() 切换类样式
  • 样式方法
  • 获取元素的方法
  1. .children() 获取当前元素的子元素，括号中标签的名字就是子元素的某个标签
  2. .next() 当前元素的下一个兄弟元素
  3. .nextAll() 当前元素后面所有的兄弟元素
  4. .prev() 当前元素的前一个兄弟元素
  5. .prevAll() 当前元素前面所有的兄弟元素
  6. .siblings() 获取当前元素所有的兄弟元素
  7. .parent() 当前元素的父级元素

  • 获取元素的方法

  • 动画的方法

  1. .show()显示 .hide()隐藏
  2. .slideDown() 滑下–> 显示
  3. .slideUp() 滑上–> 隐藏
  4. .slideToggle() 切换滑上滑下
  5. .fadeIn() 淡入—>显示
  6. .fadeOut() 淡出—> 隐藏
  7. .fadeToggle() 切换淡入淡出
  8. .fadeTo() 设置透明度 —> fadeTo(1000,0.3)
  • 以上所有方法可以只传入一个参数，数字类型：表示毫秒，也可以字符串”show” “normal” “fast” 也表示时间，也可以传入两个参数，时间，回调函数
  1. .find() 寻找当前元素对应的子元素
  2. .animate() 参数1：键值对 2.时间 3. 回调函数
  3. .stop() 停止正在进行的动画

  4. .attr() 获取或设置元素属性

    • 如果是设置： attr(“属性”,”值”)
    • 获取：attr(“属性”)

  5. .prop() 获取和设置复选框的选中状态
