php基础

文件以.php后缀结尾，所有程序包含在<?php 这里是代码 ?>
避免使用中文目录和中文文件名
php页面无法直接打开需要运行在服务器环境当中

★ 1.基本语法

  1. 必须写在<?php ?>里面，一个页面可以写多个，在一个页面定义的变量可以共用
  2. 定义变量 $
    • 变量的类型：字符串、整型、浮点型、布尔类型、数组
    • 变量以$开头 字母/数字/下划线 不能以数字开头,大小写敏感（区分大小写）
  3. 字符串链接 “.”
  4. echo 输出语句

★ 2. 数组

2.1普通数组

2.2关联数组

★3.输出函数：

  1. echo :输出字符串的
  2. print_r() 输出数组或者对象的
  3. var_dump() 输出数组里面的详细信息

★4.常见的函数

  1. count() 统计数组的长度
  2. file_get_contents() 读取文件里面的内容，读取出来的内容是字符串
  3. in_array() 判断数组当中是否存在某个元素，用于普通数组
  4. array_key_exists 判断数组当中是否存在某个key
  5. move_upload_file() 移动上传的文件

★5.客户端与服务端进行交互

5.1 客户端发送请求给服务器：

  1. 在地址栏输入地址发送请求
  2. 点击一个超链接
  3. 表单提交

    5.2 发送请求的两种方式

    发送请求是给服务器端进行提交到服务器，有两种方式

5.2.1 get

  • 在地址栏输入地址发送请求、点击超链接
  • 表单提交，默认是get
    • 接收数据：$_GET; 对应的值是一个关联数组
    • 发送参数?paraName=value&paraName=value&paraName=value

      5.2.2 post

      • 表单提交 设置为post
      • 接收数据：$_POST;

★★get 和 post的区别

  1. get提交请求的数据都在地址栏中，相对不安全
  2. get请求的数据都请求地址的后面，对请求的数据大小有限制
  3. get没有请求体，也会少一些请求头，所以它传递到服务端的数据要少一些
  4. post 请求的数据在请求体中，相对来说安全
  5. post 请求的数据大小没有限制
  6. post提交有一个特殊的请求头：Content-Type: application/x-www-form-urlencoded,可以对请求的数据包含中文的数据进行编码。
