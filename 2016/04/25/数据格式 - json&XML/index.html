二、xml

如果跟数据库交互，如果传递的数据格式非常大，非常多的情况下，我们一般给传递数据规定一个格式。这个格式有两种，一种是xml，一种是json。

2.1概念

2.1.1 xml 可扩展的标记语言

  • 标记语言：

    • xml：可扩展的标记语言
    • html：超文本标记语言
    • 关系：都是由标记标签组成。
    • 区别：
      1. xml里面的标签是自定义的，html里面的标记是w3c指定的，必须按照规范去写
      2. xml主要是用来做软件的配置文件、数据的传输，一般自己写解析；html主要是用来做界面显示的，浏览器解析
