http服务

http服务

★1.bs架构与cs架构

  • bs架构：borwser server ,基于浏览器去上网
  • cs架构：client sever 客户端，需要安装
  • 区别：
    • 性能方面：
      1. bs所有的内容都需要从服务端加载，性能稍差些
      2. cs性能相比bs好一些，有安装包，可能集成一些大型的图片
    • 安全性：
      1. cs比bs安全性更高
    • bs免安装，开发效率高

★2. 客户端与服务器

  • 客户端：具有向服务端索取服务能力的终端，安装不同的应用软件，索取不同的服务
  • 2.2服务器：对外提供服务的计算机
  • 计算机+操作系统（windows,Linux）+安装不同的服务端的软件，提供不同的服务

★3. 网络

3.1 IP

公网IP ：去电信申请，需要营业执照
局域网IP：某一区域内由多台计算机互联成的计算机组

3.2 域名

域名类型：.com .net 代表商用 .cn代表中国 .gov 政府，组织
域名申请：阿里云申请，注册
域名备案：工商局，一般一个星期

3.3 端口

  • 一台计算机有6w多个端口，计算机提供服务时必须占用一个端口对外提供服务
  • web端口默认80
    • mysql端口默认是3306

      3.4 DNS(domain name system)域名系统，进行域名管理

★4. 搭建web服务器

  1. 计算机
  2. 操作系统
  3. 应用软件 web应用软件：apache，tomcat，node.js

4.1安装wamp

  • wamp是一个集成软件，里面包含了好几个软件： apache mysql php
  • 访问自己本机的三种方式
    1. http: //ip地址:80 （没有配置apahce之前显示没有权限）
    2. 127.0.0.1 （进入wamp的欢迎界面）
    3. http:// localhost/（没有配置apahce之前显示没有权限）

4.2apache的配置

  1. 配置所有的用户都可以访问我的服务器

    • 进入到下面这个目录

      C:\wamp\bin\apache\Apache2.2.21\conf\httpd.conf

    • 在235行添加 Allow from all
    • 重启

  2. 修改wamp 的apache 里面默认访问路径，修改完成后将我们制作好的网页拷贝到配置好的根目录下，浏览器访问即可。

    • 进入到下面这个目录

      C:\wamp\bin\apache\Apache2.2.21\conf\httpd.conf

    • 文件的 178行 DocumentRoot “f:/文件夹名称”
      • 文件的205 行
    • 重启

  3. 配置通过域名可以访问到我的服务器

    • 配置域名拦截：
      • 本地的c://windows/system32/drivers/etc/hosts 文件，修改hosts文件权限
    • 打开C:\wamp\bin\apache\Apache2.2.21\conf\httpd.conf 467行的注释
    • 进入C:\wamp\bin\apache\Apache2.2.21\conf\extra找到httpd-vhosts.conf进行配置

  4. 配置完成后就可以通过www . tianmao.com在本地访问存放在f:/ajax/tianmao文件下的index文件了

★5.资源

  • 静态资源：可以通过浏览器直接打开，直接解析的我们成为静态资源，html,js,css,image
  • 动态资源：需要被服务器转换成静态资源，php，jsp，asp
  • 区别：
    1. 浏览器只认识静态资源
    2. 动态资源是在服务端运行的，静态资源是在客户端运行的
